# Training Planner

## Автор
- Имя и фамилия: Гюльмагомедов Исмаил

## Описание
`Training Planner` — GUI-приложение на Python Tkinter для планирования тренировок.

Возможности:
- Добавление тренировки: дата, тип, длительность
- Валидация:
  - дата в формате `YYYY-MM-DD`
  - длительность — положительное число
- Фильтрация по типу тренировки
- Фильтрация по дате
- Удаление выбранной записи
- Сохранение и загрузка данных через `trainings.json`

## Запуск
```bash
cd TrainingPlanner
python3 main.py
```

## Пример использования
1. Введите дату (`2026-05-04`), тип (`Бег`), длительность (`45`).
2. Нажмите `Добавить тренировку`.
3. Выберите фильтр по типу или укажите дату фильтрации.
4. Сохраните данные кнопкой `Сохранить JSON`.
